# virtual-fitting

## Environment

```
node >= 4
```

## Code Style

https://github.com/airbnb/javascript

## Develop

1. ```
   cd assets/ && python simple-cors-http-server.py
   ```

2. ```
   npm start
   ```

3. ```
   python api.py
   ```

4. ```
   python imgdb.py
   ```

5. ```
   mongod
   ```

visit http://127.0.0.1:8989

npm run build
